Molten Client

A custom Minecraft Java Edition launcher

Features

  • Microsoft Authentication - Sign in with your Microsoft account
  • Mod Manager - Browse, install, enable/disable, and delete Fabric mods via Modrinth
  • Resource Packs and Shaders - Manage and download from Modrinth
  • Skin Viewer - 3D skin preview
  • Screenshots - Browse and manage your Minecraft screenshots
  • Console - View live Minecraft logs
  • Crash Reports - Read crash reports directly in the launcher
  • Profiles - Multiple launch profiles
  • Server Quick Join - Jump straight into a server from the launcher

Built With

Tauri React TypeScript Rust Vite
