# Molten Client

Molten Client - Minecraft Launcher

  A custom, personal Minecraft Java Edition launcher built with Tauri, React, and Rust.

## Features

  - **Microsoft Authentication** — Sign in with your Microsoft account, currently being reviewed, HELLO !
  - **Mod Manager** — Browse, install, enable/disable, and delete Fabric mods via Modrinth api, using .disabled for disabling mods. also can install modpacks from modrinth
  - **Resource Packs & Shaders** — Manage and download from Modrinth 
  - **Skin Viewer** — 3D skin preview, install skins from username, view and select slim or normal
  - **Screenshots** — Browse and manage your Minecraft screenshots from your folder directly in the app
  - **Console** — View live Minecraft logs, see errors, etc
  - **Crash Reports** — Read crash reports directly in the launcher
  - **Profiles** — Multiple profiles, can customize a profile for survival, pvp, skyblock, etc
  - **Server Quick Join** — Jump straight into a server from the launcher, the launcher will force join you to the server, directly after minecraft has launched
